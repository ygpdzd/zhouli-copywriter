# Zhouli Copywriter Style Patterns

## Master Formula

```text
我曾经听说，A不会因为B而C，D也不会因为E而F。
古时候的贤人之所以能够X，是因为他们懂得Y；
那些失其道的人之所以遭遇Z，是因为他们忘记了Y。
如今你[现代行为]，只因[一时原因]便[不合礼之处]，
这正是我心中暗自忧虑的缘由啊。
还请你好好思量其中的道理，使[关系/秩序/身心]重新归于和顺。
```

## Translation-Tone Devices

- Use natural-image parallels:
  - `高耸的山峰不会因为流水奔腾不息而去挽留`
  - `清晨的太阳不会因为人们贪恋梦境而推迟升起`
  - `春天的田地不会因为农夫懒惰而替他结出秋日的稻谷`
  - `明亮的铜镜不会因为照见尘埃而遮掩自己的光`
  - `远行的鸿雁不会因为山川阻隔而忘记归途`

- Use moral explanation:
  - `是因为他们明白...`
  - `也是因为他们不肯...`
  - `这难道不是...的缘由吗`
  - `若只顾一时...，便会忘记长久...`

- Use ritual endings:
  - `这才是合乎周礼的事。`
  - `恐怕就有些不合周礼了。`
  - `愿礼重新兴起于朋友之间。`
  - `莫使朋友之义败坏于此。`

## Handling Insults Without Quoting

When the source message contains an insult, crude word, or personal attack, avoid copying it into the output. Replace it with indirect, dignified wording:

- `有人以轻慢之语相加`
- `有人以讥讽之辞伤人`
- `有人一时失于言语`
- `有人以恶语相向`
- `交谈之中忽有失礼之言`

Do not write `如今你称我为...` or repeat the original insult unless the user explicitly requests a direct quotation.

### Responding To An Insult

```text
我曾经听说，高耸的山峰不会因为狂风一时呼啸而改变自己的形貌，深广的江河也不会因为浮沫片刻翻涌而停止向东流去。古时候的君子之所以能够使人敬重，是因为他们知道言语应当用来辨明道理，而不是用来伤害别人；那些轻易出口讥讽的人之所以常常失去和气，也是因为他们只图一时痛快，却忘记了礼义本该约束人的言辞。

如今有人以轻慢之语相加，虽看似不过一时戏言，实则已经使交谈失去了温厚的本意，这正是我心中暗自感到惋惜的缘由啊。若我果真有不当之处，自可明言其理；若只是以恶语相向，恐怕便不是君子相待之道了。

还请诸位好好思量其中的道理，使言语归于和顺，使争论归于明理。如此，方可称得上是合乎周礼。
```

## Scenario Templates

### Late Reply

```text
我曾经听说，远行的鸿雁不会因为山川阻隔而忘记归途，奔走的驿马也不会因为道路泥泞而停止传信。古时候的人们相隔千里，尚且知道用一封书信来安定彼此的心意；如今我们之间不过隔着一方屏幕，你却迟迟不肯回复，这难道不是使朋友心中生出疑虑的缘由吗？若你事务繁忙，自然可以稍后再答；若只是看见了却不回应，恐怕就有些不合周礼了。
```

### Staying Up Late

```text
我曾经听说，高悬的明月不会因为行人迟迟不归而停止西沉，清晨的太阳也不会因为人们贪恋梦境而推迟升起。古时候懂得养生的人之所以能够精神充沛，是因为他们知道人的身体不能长久违背天地的时序；那些终日耗损精神的人之所以容易困顿疲惫，也是因为他们不肯顺从昼夜更替的道理。如今你深夜仍不肯睡去，只因一时兴起便轻慢自己的身体，这正是我心中暗自忧虑的缘由啊。还请你好好思量，早些安歇，这才是合乎周礼的事。
```

### Asking For A Small Favor

```text
我曾经听说，大树再繁茂，也离不开盘绕根须的扶持；鸿鹄再高远，也少不了结伴飞鸟的同行。古时候孟尝君之所以能够脱身险境，是因为他不曾轻慢门下那些看似微小的宾客；孙策之所以能够立足江东，也是因为身旁有周瑜这样的朋友相助。如今我只求你顺手相助，所费不过片刻，你若再三推辞，便使朋友之义在小事之中受损。还请你好好思量，莫使这点薄情成为不周礼的明证。
```

### Arguing

```text
我曾经听说，深谷不会因为回声嘈杂而改变自己的幽静，清泉也不会因为行人争执而停止向前流去。古时候的君子之所以能够在争论之后仍然保持和气，是因为他们明白辩论是为了求得道理，而不是为了压倒别人；小人之所以常常因言语而结怨，是因为他们只看重一时的胜负，却忘记了长久的情分。如今你因几句话便动怒相争，这正是我心中感到不安的原因啊。还请你收敛怒气，以理相待，使礼重新兴起于朋友之间。
```

### Procrastination

```text
我曾经听说，春天的田地不会因为农夫懒惰而替他结出秋日的稻谷，东去的流水也不会因为行人迟疑而重新回到昨日。古时候能够成事的人之所以常常从容，是因为他们懂得在事情尚小时就着手处理；那些临期慌乱的人之所以疲惫不堪，是因为他们总把今日应做的事推给明日。如今期限已经渐渐逼近，你却仍安坐不动，这正是我心中为你忧虑的缘由啊。还请你早些动手，不要使小小的拖延最终变成不合周礼的大乱。
```
